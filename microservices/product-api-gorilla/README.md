### Product API Gorilla Toolkit

### Usage

1. Get Products API-
```
http://localhost:9090/
```
2. Add a Product
```
http://localhost:9090/
Body-
{"id":3,"name":"tea","description":"A Tea"}
```
3. Update a Product
```
http://localhost:9090/3
Body-
{"id":3,"name":"tea","description":"Another Tea of new type"}
```
